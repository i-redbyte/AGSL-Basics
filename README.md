# Основы AGSL для android разработчика

Этот репозиторий содержит полный исходный код примеров, использованных в статье:

**"Основы AGSL для Android-разработчика"**

Здесь собраны 3 самостоятельных демонстрационных примера, показывающих, как использовать **AGSL (
Android Graphics Shading Language)** в связке с **Kotlin** и **Jetpack Compose** для создания
процедурных визуальных эффектов прямо в UI без OpenGL/Vulkan.

## Содержание

- **01 — Анимированный градиент**  
  Простейший пример RuntimeShader: фон, который "дышит" за счёт синус-градиентов и uniform-параметра
  `u_time`.

- **02 — Волна от тапа (Ripple)**  
  Реактивный эффект: волна расходится от координаты касания, затухает по времени и радиусу.
  Используются uniform-ы `u_touch`, `u_time`, `u_active`.

- **03 — Летающий объект по параболе (звезда)**  
  Пятиконечная звезда летит по параболе. Вся математика траектории - в Kotlin, форма и заливка - в
  AGSL. Во время полёта кнопку нельзя нажать повторно.
